1. TextBlock.razor OK
•	string Text : contenu texte (HTML ou Markdown)
•	string? Title : titre optionnel
•	string? Style : style ou classe CSS

2. ImageBlock.razor OK
•	string Url : chemin ou URL de l’image
•	string? Alt : texte alternatif
•	string? Caption : légende
•	string? Style : style ou classe CSS

3. VideoBlock.razor
•	string Url : URL de la vidéo (YouTube, Vimeo, fichier)
•	string? Caption : légende
•	bool Autoplay : lecture automatique
•	bool Controls : afficher les contrôles

4. AudioBlock.razor OK
•	string Url : URL ou chemin du fichier audio
•	string? Caption : légende
•	bool Autoplay : lecture automatique
•	bool Controls : afficher les contrôles

5. QuizBlock.razor OK
•	string Question : texte de la question
•	List<string> Choices : propositions de réponse
•	int? CorrectIndex : index de la bonne réponse
•	string? Explanation : explication ou feedback

6. LinkBlock.razor OK
•	string Url : URL du lien
•	string? Text : texte affiché
•	string? Description : description ou info-bulle

7. TableBlock.razor
•	List<List<string>> Rows : données du tableau (lignes et colonnes)
•	List<string>? Headers : en-têtes de colonnes

8. FileBlock.razor OK
•	string FileName : nom du fichier
•	string Url : URL ou chemin du fichier
•	string? Description : description ou instructions

9. SpaceBlock.razor 
•	int Height : hauteur de l’espace (en px ou rem)

10. InfoBlock.razor OK
•	string Message : texte d’information
•	string? Type : type d’alerte (info, warning, success, etc.)
